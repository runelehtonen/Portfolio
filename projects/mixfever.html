MIXFEVER

Project Description (3. Semester - Group project)

Mixfever was our first project on 3rd semester. We developed a single page application that enables the user to search for drinks based on ingredients they already own. This was made possible by structuring all cocktails and ingredients in JSON-format, which we later appended all over the app with vanilla JS. The web app was developed with mobile-first in mind but works on all devices.
